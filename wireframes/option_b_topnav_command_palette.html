Option B: Top Nav + Command Palette — Key design decisions:
  • Flat top nav with minimal primary links — the nav bar has only the top-level sections. Sub-pages are reached via the page itself or the command palette. Maximizes vertical content space.
  • Command palette (⌘K) — a Stimulus controller provides instant fuzzy search across all 72 admin routes. Power users never need to click through menus. Backed by a JSON index endpoint.
  • Turbo Frame tabs — switching between "All / Active / Draft / Low Stock" swaps just the table content via a <turbo-frame>, no full page load.
  • Slide-over panel for quick create/edit — "Add Product" opens a slide-over (Stimulus controller) whose form submits via Turbo Stream, appending the new row to the table without navigation.
  • Search filter with debounce — a Stimulus controller auto-submits the filter form after 300ms, fetching a Turbo Frame update for the table.
  • Content-centric layout — generous whitespace, max-width container, no sidebar eating screen space. Great for data-heavy list views.
  • Uses Propshaft + plain CSS — no build step, no Tailwind, no Node.js. Pure Rails 8 asset pipeline.
