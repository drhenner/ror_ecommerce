Option B: Magazine / Category-Storytelling Storefront — Design decisions & implementation notes:
  • Transparent nav overlaying hero that transitions to solid on scroll. A Stimulus controller watches IntersectionObserver on the hero and toggles .solid class. Maps to existing _welcome_header.html.erb.
  • Mega-menu on hover for category links — each panel shows featured category images. Uses product type images from ProductType and brand logos from Brand.
  • Asymmetric editorial grid mixes a large featured product (2x2 span) with smaller cards. The large card uses Product.featured, the rest from recent products. All link to products#show.
  • Category stories blend editorial content with shopping. These could be backed by a simple CMS model or static partials initially. Each links to filtered products_path(product_type_id: ...).
  • Brand badges on product cards — leverages existing product.brand_name method.

    • Product Listing Page

  • Masonry-style grid with 2-4 columns. Interspersed editorial banners break up the grid every 8-12 products. Sort and filter controls as a horizontal bar (not sidebar). Uses Turbo Frames for filter/sort to reload only the grid.

    • Product Detail Page

  • Full-width hero image at top, product details in a centered card below. Image gallery as horizontal scrollable strip. Variant selection + add to cart in a sticky bar that appears on scroll.

    • Cart Page

  • Full dedicated page (not drawer) with rich product cards showing images, variant details, and editorial-style layout. Order summary panel on the right. No tables.

    • Checkout & My Account

  • Checkout: centered single-column with steps as a horizontal progress bar. Clean form fields with inline validation.
  • My Account: horizontal tab navigation across top (Orders, Addresses, Cards, Profile, Referrals, Credit).

    • Tech Stack

  • Custom CSS with CSS custom properties for theming. No framework dependency.
  • Turbo Drive for smooth page transitions, keeping nav and footer mounted.
  • Turbo Frames for product grid filtering, cart updates, variant selection.
  • Stimulus for: transparent/solid nav toggle, mega-menu, image galleries, sticky add-to-cart bar.
  • Existing controllers and routes unchanged.
