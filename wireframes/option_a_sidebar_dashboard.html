Option A: Sidebar Dashboard — Key design decisions:
  • Fixed sidebar with collapsible nav groups — always visible, reduces clicks to reach any section. Would use a Stimulus controller for collapse/expand + persist state in localStorage.
  • Turbo Drive replaces full page reloads — the sidebar stays mounted, only <main> content swaps. Zero custom JS for navigation.
  • Turbo Frames for independently-refreshable panels (recent orders auto-poll every 30s). Each panel loads/updates without touching the rest of the page.
  • Turbo Streams over WebSocket (Action Cable) for real-time activity feed — new orders and stock alerts push instantly.
  • Command palette (⌘K) via a Stimulus controller for power-user quick navigation across all 72 admin controllers.
  • CSS uses Propshaft + CSS custom properties — no Tailwind dependency, easy theming, works with Rails 8 defaults.
  • All existing admin controllers/routes are unchanged — this is purely a layout + view layer modernization.
